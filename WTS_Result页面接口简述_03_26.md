## Params

- `op` : 操作(即这是什么操作的返回结果)
- `type` : 结果类型
	- 0	: 失败
	- 1 : 成功
	- 88 : 非微信访问
	- 其他 : 非法访问
- `msg` : 自定义消息
- `title` : 自定义标题
- `icon` : 自定义图标样式
	- [Weui Icon Wiki](https://github.com/weui/weui/wiki/Icon)
- `btn` : 自定义按钮(hide:表示隐藏,其他表示指定文本) 
- `redirect` : 3秒后跳转的页面URL
